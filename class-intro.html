    • 地暖運作原理與方式

      1. 發熱体經由管壁散發熱量(傳導)到週遭覆蓋體(蓄熱層)
      2. 發熱体下方設置隔熱層，斷絕熱量往下流失而祗往上傳遞
      3. 空氣分子將熱量傳遞給其它空氣分子(對流) ，高度愈低的 空氣接收的熱量愈多
      4. 空氣分子覆蓋人體而將熱量傳導至人體內，同時蓄熱層亦將熱量傳導給人體最冷的足底

    • 地暖分類

      • 水地暖

      藉由水管內的溫水傳遞熱量
      水暖管種類:
      1. PERT
      2. PEX
      3. 含阻氧膜層三層式(PEX或/和PERT複合管)
      4. 含阻氧鋁片層三層式(PEX或/和PERT鋁塑管)

      電地暖

      透過電阻將電能轉換成熱能
      電阻種類:
      1. 合金：合金電纜(**)、合金電熱膜
      2. 碳素：碳素電纜、碳素電熱纜
      3. 碳墨高分子化合物：自限溫電纜、自限溫電熱膜
      4. 碳墨印刷：一般電熱膜(*)、PTC電熱膜(*)
